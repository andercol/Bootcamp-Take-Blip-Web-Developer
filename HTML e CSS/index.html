A tag de links <a> </a> é utilizada para criar hiperlinks.

Exemplo:

A tag "<a href="https://www.linkedin.com/in/andersoniscolin/">Meu linkedin</a>" - irá gerar o resultado: Meu linkedin

A tag "<a href="mailto:andercol@gmail.com">Meu e-mail</a>" - irá gerar o resultado: Meu e-mail

A tag "<a href="https://www.linkedin.com/in/andersoniscolin/" target="_blank">Link</a>" - irá gerar o resultado: Link
